# Pixeltable Documentation (Docusaurus)

This is the Docusaurus version of the Pixeltable documentation.

## Development

```bash
cd docs3
npm install
npm start
```

## Build

```bash
npm run build
```

## Deploy

```bash
npm run deploy
```
