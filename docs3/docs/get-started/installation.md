# Installation

Get Pixeltable up and running in under 2 minutes with pip.

## Prerequisites

- Python 3.9 or higher
- pip package manager

## Install Pixeltable

```bash
pip install pixeltable
```

## Verify Installation

```python
import pixeltable as pxt
print(f"Pixeltable version: {pxt.__version__}")
```
