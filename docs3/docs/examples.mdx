## Manual Examples (For Now)

While we set up the interactive environment, here are some key Pixeltable examples:

```python
# Install Pixeltable
pip install pixeltable

# Import and create a table
import pixeltable as pxt

t = pxt.create_table('demo_table', {'name': str, 'age': int})

# Add data
t.insert([
    {'name': 'Alice', 'age': 25},
    {'name': 'Bob', 'age': 30}
])

# Query data
result = t.select(t.name, t.age).collect()
print(result)
```
