# Core API

The main Pixeltable module provides the fundamental functionality for creating and managing tables, views, data types, and operations.

## Table Operations

### create_table

```python
create_table(path: str, schema: Optional[dict[str, Any]] = None, *, source: Optional[TableDataSource] = None, ...)
```

Create a new base table with optional data import from various sources.

---

### create_view

```python
create_view(path: str, base: Union[Table, DataFrame], *, additional_columns: Optional[dict[str, Any]] = None, ...)
```

Create a view of an existing table object with optional additional columns.

---

### create_snapshot

```python
create_snapshot(path: str, base: Union[Table, DataFrame], *, additional_columns: Optional[dict[str, Any]] = None, ...)
```

Create a snapshot of an existing table object.

---

### create_replica

```python
create_replica(destination: str, source: Union[str, Table])
```

Create a replica of a table, either remote or local.

---

### get_table

```python
get_table(path: str)
```

Get a handle to an existing table, view, or snapshot.

---

### drop_table

```python
drop_table(table: Union[str, Table], force: bool = False, if_not_exists: Literal["error", "ignore"] = "error")
```

Drop a table, view, or snapshot.

---

### list_tables

```python
list_tables(dir_path: str = "", recursive: bool = True)
```

List the tables in a directory.

---

## Directory Operations

### create_dir

```python
create_dir(path: str, if_exists: Literal["error", "ignore", "replace", "replace_force"] = "error", parents: bool = False)
```

Create a directory.

---

### drop_dir

```python
drop_dir(path: str, force: bool = False, if_not_exists: Literal["error", "ignore"] = "error")
```

Remove a directory.

---

### list_dirs

```python
list_dirs(path: str = "", recursive: bool = True)
```

List the directories in a directory.

---

### ls

```python
ls(path: str = "")
```

List the contents of a Pixeltable directory.

---

### move

```python
move(path: str, new_path: str)
```

Move a schema object to a new directory and/or rename it.

---
